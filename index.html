Programme Python - Calcul de la Factorielle

Voici un programme simple qui calcule la factorielle d'un nombre :

Code :

def factorielle(n):
    if n == 0:
        return 1
    else:
        return n * factorielle(n - 1)

# Exemple d'utilisation
nombre = 5
resultat = factorielle(nombre)
print(f"La factorielle de {nombre} est {resultat}")

Comment l'utiliser :

  1. Copia le code ci-dessus dans un fichier nommé factorielle.py.
  2. Exécute le fichier en utilisant la commande : python factorielle.py.
