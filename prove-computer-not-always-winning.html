Preuve : L'ordinateur n'est pas toujours gagnant

Le jeu Pierre-Papier-Ciseaux est basé sur un algorithme aléatoire qui ne favorise ni le joueur, ni l'ordinateur. Voici un extrait de code simple qui montre que le choix de l'ordinateur est totalement aléatoire : 

        
        // Choix de l'ordinateur, tiré aléatoirement entre 🪨, 🍁 et ✂️
        const choices = ['🪨', '🍁', '✂️'];
        const computerChoice = choices[Math.floor(Math.random() * choices.length)];

Comme vous pouvez le voir, chaque fois que l'ordinateur joue, il sélectionne aléatoirement une des trois options disponibles. Il n'y a pas de biais systématique pour favoriser l'ordinateur. À chaque tour, il est tout aussi probable pour l'ordinateur de perdre, de gagner ou de faire égalité.

Voici un autre exemple de code qui montre la logique de décision du jeu pour déterminer le gagnant : 

        
        if (playerChoice === computerChoice) {
            result = 'Égalité !';
        } else if (
            (playerChoice === '🪨' && computerChoice === '✂️') ||
            (playerChoice === '🍁' && computerChoice === '🪨') ||
            (playerChoice === '✂️' && computerChoice === '🍁')
        ) {
            result = 'Vous gagnez !';
        } else {
            result = 'Vous perdez !';
        }

Ce code montre que le résultat du jeu est basé sur les règles classiques de Pierre-Papier-Ciseaux, et la victoire n'est pas prédéterminée par un biais en faveur de l'ordinateur.
